This is my attempt at a Fractal Notepad app.
It features:
  The ability to create your own notes! _(Wow, shocker)_
  The ability to draw on your notes
  Edit both the note text and the title of the note
  Create notes _inside_ other notes
The apk can be found here: https://drive.google.com/file/d/1E7ByYKQXgYqCkvHzUVEWVcS4PL2moV4T/view?usp=sharing
